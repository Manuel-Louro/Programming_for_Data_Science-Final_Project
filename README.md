# Where Should I Live? - Programming for Data Science Final Project
This project was developed as my final assignment for the **Programming for Data Science** course at **Nova IMS**.

The objective of this project was to analyze a dataset containing information about European cities and extract meaningful insights to help individuals identify the most suitable city to live in based on various quality-of-life factors.

The script begins with a comprehensive data cleaning and preprocessing phase, ensuring the dataset is reliable for analysis. It then performs exploratory data analysis, generating valuable insights and statistical visualizations that highlight key characteristics of each city.

In addition, the project features an interactive map of Europe, allowing users to easily explore essential information about each city in a visual and intuitive way.

To further enhance the user experience, the project includes a recommendation interface that helps users identify the European city that best matches their preferences. Users can either select from predefined lifestyle profiles or customize the importance of different quality-of-life factors, enabling personalized recommendations tailored to their individual priorities.
